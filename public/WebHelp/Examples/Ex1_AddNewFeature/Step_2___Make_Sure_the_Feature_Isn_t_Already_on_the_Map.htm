Step 2: Make Sure the Feature Is Not Already on the Map

 

Before I add this museum, I want to make sure it is not already on the map.   

 

Here is how I check to see if the Astor House Museum is not already on the map:

 

(a)  Search for the feature using the online editor.

 

  • I enter "Astor House, Golden, CO" in the "Search" bar of the online editor and click the "Go" button:

 

 

  • If no results come up in the online editor search engine search for it using a map search engine.

 

  • In this case, there is a record of its general location in OpenStreetMap (top search result area).  This does not mean that the feature is already on the map. If there is a result in GeoNames (bottom search result area), the feature might already be on the map.

 

  • Next, I click on the record in the OpenStreetMap search result area to zoom to the general location of where the Astor House Museum is located.

 

  • Then, I look for a Museum icon on the map.  

 

  • In this case, there are not any Museum icons in this area.  

 

  • To confirm that the Astor House is not already on the map, I look for other features in the area that could be related to a Museum.  For example, a Museum could be incorrectly labeled on the map as a Historic Site / Point of Interest.

 

      • To check to see if the Astor House Museum is already on the map, but incorrectly labeled as another type of feature, I select an icon already on the map by clicking on the feature.  

 

      • Then, I check to see if the name in the "Name" field is "Astor House"; the same name as the feature I want to add to the map.

 

  • In this case, there are not any features on the map with the name "Astor House".  So I can proceed to the next step in adding a new feature to the map.

 

  • Note, if I found that the Astor House Museum was already on the map, but labeled incorrectly as a Historic Site / Point of Interest feature, I would change the type of feature by selecting the correct feature from the menu above the "Name" field.
