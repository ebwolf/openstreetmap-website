User Guide for Contributing Structure Data to The National Map

Version 1.1 (last updated June 18, 2012)

 

 

The USGS is currently testing procedures for looking for volunteers to collect and update USGS geographic data. Similar to how OpenStreetMap allows anyone to collect, edit, and use geographic data through an online map editor, the USGS has developed a prototype online editor customized to our data to allow volunteers to contribute data to The National Map and The National Structures Dataset. 

 

We are looking for people like you to work with us to collect 20 types of structures for the USGS. The data you will collect during this project will be loaded into The National Map.  If you have access to the Internet and are willing to dedicate some time editing map data we hope you will consider participating!

 

Source:  http://viewer.nationalmap.gov/viewer/

 

WHAT WOULD I BE DOING IF I PARTICIPATE?

 

You would be using our online editor to edit USGS structures map data by updating existing features and adding new features. You can choose what types of structure features to edit, (from our list of 20) to edit anywhere in Colorado, the 2012 pilot project area!

 

 

HOW DO I GET INVOLVED?

 

1)      Read through this document to learn more about the project and how to contribute.

 

2)      Create a Potlatch online map editor account.

 

3)    Follow the directions on how to edit and add points to get started.
