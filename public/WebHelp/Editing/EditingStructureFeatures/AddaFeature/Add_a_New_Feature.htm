How to Add a New Feature

Overview

 

To add a structure feature, add an icon to the map for each unique building use, not an icon for each building.

 

Example:  One building (city hall) with multiple uses (police station, fire station, and city hall) is identified on the map by adding an icon for each unique use to the middle of the building.

 

 

 

Steps for Adding New Structure Features

 

STEP 1:  Find a Structure Feature to Add that is Not Already on the Map

 

  • See "Resources" and "How to Research a Feature" for tips on finding structure features to map.

 

  • You may need to zoom in to confirm that your feature is not already on the map.  The structure icons in the online editor may overlap with one another if you are not zoomed in close enough.  

 

STEP 2:  Place an Icon on the Map

 

  • Drag the appropriate structure icon from the menu on the left to the structureâ€™s location on the aerial imagery.

 

  • Be sure to zoom in close on a building before adding the structure icon to the map.  

 

STEP 3:  Enter Information About the Feature

 

    • While the icon is still selected, enter the structure feature's name in the "Name" field on the "Required" tab:

 

 

Note:  Please use the full proper name as it is found on an official website.

 

    • You may also fill out address information for the structure feature under the "Optional" tab, but this information is not required:

 

 

    • When editing names and addresses it is not acceptable to use general internet websites.

 

    • Acceptable ways to locate name and address information include:

 

      • Website of the feature (e.g., a school website, a city's website of libraries, etc.).

 

      • Phone call to confirm the name and address of the feature.

 

      • Local knowledge (e.g., you already know the name of the elementary school or library in your neighborhood).

 

      • Visual confirmation (e.g., in person or using online maps with "on-the-ground" views of the street).

 

 

STEP 4:  Click the "Finished Editing" box under the "Required" tab

 

  • When you are done adding the structure feature, click the "Finished Editing" box located under the name of the feature:

 

 

 

    • THIS STEP IS VERY IMPORTANT:

 

    • Click the "Finished Editing" box if you looked at an existing point and didn't change the feature's information (name, address, or type of feature) because the information was correct.

 

    • Click the "Finished Editing" box if you looked at an existing point and made changes to the feature's information (name, address, or type of feature).

 

  • Note:  The icon for the structure feature will now have a green outline,  which lets us know you have finished that point: 

 

 

 

STEP 5:  SAVE YOUR CHANGES!!

 

  • Make sure to save after finishing adding a structure feature!
