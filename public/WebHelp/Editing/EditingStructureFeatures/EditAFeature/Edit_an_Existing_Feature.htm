How to Edit an Existing Feature

 

Overview

 

Existing structure features that have not been edited will have a red border in the online map editor.  

                   

 

To edit existing structure features, check the feature's location and information (name, address, and type of feature).

 

 

Steps for Editing an Existing Structure Feature

 

STEP 1: Check Location

 

(a) Using the Aerial Imagery, visually check to see if the structure icon appears to be on the correct building.     

 

                                

  • In the photo, above, we can see that the icon is correctly located on the state capitol building.

(b)  To verify the point’s location, use official websites and additional image services.

 

    • In the fire station example, above, a name and address are already available for the feature.  After searching the name we find an official website—in this case the City of Denver website which lists fire stations.  

 

    • Looking at the photo provided on the website (see below), we can see that the fire station matching the name and address listed for the feature appears to be the same as the fire station in the image.

STEP 2: Check the Feature's Information

 

  • To check an existing feature's information (name, address, and type of feature), click on the feature to select it.  The feature's information will appear on the left side of the imagery.  The only required information is the structure feature's "Name".  Please use the full proper name as it is found on an official website.  

 

  • You may also check the address information listed in the "Optional" tab, but it is not required.

 

  • When editing names and addresses, it is not acceptable to use general internet websites.  Use the most official source you can, such as a school website, a city’s website containing library listings, etc.  

 

  • Acceptable ways to locate name and address information include:

 

    • Website of the feature (e.g., a school website, a city's website of libraries, etc.).

 

    • Phone call to confirm the name and address of the feature.

 

    • Local knowledge (e.g., you already know the name of the elementary school or library in your neighborhood).

 

    • Visual confirmation (e.g., in person or using online maps with "on-the-ground" views of the street).

 

  • Check that the structure feature is labeled as the correct type of feature (e.g., library, hospital, etc.).

 

    • For example, a "College / University" structure feature might be labeled as a "School" structure feature.  If this is the case, the type of feature should be corrected by clicking on the "School" icon and selecting "College / University" for the type of feature from the list above the "Name" field.  

 

When determining the correct type of feature, please refer to the feature definitions.

 

 

  • If you find a feature that no longer exists, just select it, and then click the "delete" button.

 

 

STEP 3:  Click the "Finished Editing" box under the "Required" tab

 

  • When you are done checking the structure feature, click the "Finished Editing" box located under the name of the feature:

 


 

  • THIS STEP IS VERY IMPORTANT:

 

  • Click the "Finished Editing" box if you looked at an existing structure feature and didn't change the feature's information (name, address, or type of feature) because the information was correct.

 

  • Click the "Finished Editing" box if you looked at an existing structure feature and made changes to the feature's information (name, address, or type of feature).

 

 

  • Note:  The icon for the feature will now have a green outline, which lets us know you have finished that structure feature: 

 

 

 

STEP 4:  SAVE YOUR CHANGES!!

 

  • Make sure to save after you finish checking or editing a structure feature!
