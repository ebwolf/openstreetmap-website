Getting Started

You will need to do three things to get started:

 

 

1.     Read this User Guide

 

 

2.     Create an Account

 

Before editing structure features, you will need to create an account to access the online map editor, Potlatch.

 

For step-by-step instructions on how to create a Potlatch account, see:

 

Instructions for Creating a Potlatch Account

 

 

3.   Explore the Online Editor

 

Once you have created your Potlatch account, please take some time orienting yourself to the tabs in the online editor.

 

For a introduction to the online editor, see:

 

Overview of the Online Editor

 

 

If you have any questions or run into problems getting started, please e-mail us at:  nationalmapcorps@usgs.gov.
