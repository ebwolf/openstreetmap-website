Overview of the Online Editor

The online map editor we are using for this project is Potlatch; a web-based editor developed by OpenStreetMap (http://www.openstreetmap.org) for editing maps.

 

To access the online editor, follow these steps:  

 

STEP 1:  Navigate to:  http://navigator.er.usgs.gov/.

 

STEP 2:  Log in using the button in the top right corner.

 

Note:  You need a Flash Player to use the online editor, Potlatch.  You can download Flash Player from Adobe.com.

If you are having difficulty accessing the online editor, try updating to the most recent version of Flash Player.

 

You will notice there are two tabs at the top of the online editor:  "Edit" and "View".
