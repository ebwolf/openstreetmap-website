Edit Tab

The "Edit" tab is where you can edit existing structure features and add new structure features.  

 

The "Edit" tab also displays map icons for the 20 structure features, existing structure features, and the selected background (aerial imagery or map).

 

This image explains the contents of the "Edit" tab:

 

 

The icons to the left of the imagery represent each of the 20 structure features we are collecting.

 

  • The "Background" dropdown list at the top right corner of the online editor allows you to select from a variety of imagery and map options:

 

Note:  "Aerial Imagery" MUST be used for final structure placement.

 

  • All structure features show up as icons on the map.  The red border indicates the structure has not been edited:

 

 

  • To place a new structure on the map, simply drag the appropriate icon onto the imagery.

 

  • You can toggle between the "Edit" tab and the "View" tab while editing.  However, if you have been editing, BE SURE TO SAVE before leaving the "Edit" tab because any unsaved changes will be lost.

 

  • When an icon on the map is selected, information about the structure feature (such as the name of the feature and the type of feature) appears to the left of the imagery:  

 


 

  • To change the type of feature, for example from a "School" to a "College / University", click on the icon so that the structure feature is selected, then change the type of feature by clicking on the drop-down menu above the "Required" tab and select the correct type of feature (e.g., College/University):

 

 

  • If you make a mistake, click the "Undo" button at the top, left of the imagery.

 

Note: The undo button will not work on saved edits.
